# BTDToolbox-Continued
BTDToolbox was discontinued, so I downloaded a master of the old code, fixed a large bunch of issues, made it able to work with modern NKH5, and now it's here.

## A Short History
As a way to make BTD5 modding easier, BTDToolbox was made by the creators DisabledMallis, gurrenm3, and TheSubtleKiller under the name "TD Toolbox" on github.
They then uploaded the code to github as an open source project.
Unfortunately, this was one of the cases where a great tool has very sloppy and hard to read code.
This ultimately resulted in many people adding code that was useless or extra, and also caused the project to shutdown.
This project is an attempt to salvage that old code and keep as much of it as it used to be, but also have as much new compatibility as possible.
The tutorials are (Hopefully) still valid, though if they are not, I will make my own tutorial series and link those.

## NKHook5 Support
The original BTDToolbox "supported" NKHook. Unfortunately, this old version by Mallis, while still around under a different name was discontinued.
The new one was called "NKHook5", and is now under the "NKHook" profile on github.
As NKHook5 hooks directly into the game, BTD Toolbox 1.6 now allows you to either launch the game normally and run the mod, or just replace the jet.

## Why BTD Toolbox 1.6?
The last update made to BTD Toolbox was known as "The biggest update yet", and that was update 1.5, so I wanted to just continue updates as if it wasn't discontinued.
If you'd like to contribute, make sure that you first remove all Marks of Web from the files.
